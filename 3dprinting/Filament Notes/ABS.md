# ABS Tips and Tricks
## Prevent Warping
* Use an unheated build chamber
* Set bed temp to 110C to improve adhesion
## Prevent overhang curling
* At 10mm open build chamber to reduce heat buildup in plastic and improve cooling
   Note: Try 5mm
